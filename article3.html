Pentingnya HTML dalam Web Development

Baru saja di kuliah semester 4, saya mempelajari dan menerapkan HTML untuk membuat website personal ini 100% menggunakan HTML murni (sebelum ditingkatkan dengan CSS & JS). Tidak ada framework atau library eksternal – pure HTML dengan styling HTML attributes dan layout table.

Pengalaman pertama ini membuka mata saya betapa powerful-nya HTML sebagai pondasi web. Setiap elemen di homepage ini dibuat manual pakai tag HTML sederhana.

Tag HTML yang Dipakai di Website Ini:

Tag Penggunaan
, ,
Layout keseluruhan (100% table-based)
, Styling teks & bold
Semua link navigasi
Gambar artikel & profil

Menggunakan 100% HTML membuat website ini ringan, cepat loading, dan kompatibel semua browser. Saya belajar bahwa simplicity HTML lebih powerful daripada over-engineering. Project ini bukti praktik langsung dari kuliah!
